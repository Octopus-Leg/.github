## 👋 팀 소개
안녕하세요! 문어발 확장에서 따온 **문어발🐙** 팀입니다. 저희는 백엔드 기술과 아키텍처를 깊이 있게 탐구하고 습득하는 것을 목표로 하는 스터디 그룹입니다. 다양한 백엔드 기술 스택을 다루며, 실무에 적용 가능한 지식과 경험을 쌓아가고 있습니다.

## 🚀 비전과 목표
- 백엔드 분야의 핵심 서적 `8권` 이상을 체계적으로 학습
- 백엔드 개발의 핵심 역량 강화
- 실무에서 활용 가능한 기술 스택 습득
- 팀원 간 지식 공유 및 협업 능력 향상

## 📚 스터디 운영 방식
- **모임 주기**: 주 1회
- **장소**: 온라인
- **진행 방식**: 
  - 학습한 내용 공유
  - 모의 면접 및 질의응답


## 🗺️ 학습 로드맵
1. **[면접을 위한 CS 전공지식 노트]**
   - 기간: 2024.12 ~ 2025.01
   - 주요 내용: 운영체제, 데이터베이스, 자료 구조, 네트워크

2. **[이것이 자바다(3판)]**
   - 기간: 2025.01 ~ 2025.03
   - 주요 내용: 클래스, 인터페이스, 상속, 제네릭, 멀티 스레드

3. **[스프링 교과서]**
   - 기간: 2025.04 ~ 2025.06
   - 주요 내용: 스프링 컨텍스트, 스프링 AOP, 스프링 부트, 스프링 MVC, 데이터 영속성

4. **[자바 ORM 표준 JPA 프로그래밍]**
   - 기간: 2025.07 ~ 2025.12
   - 주요 내용: JPA 기초, 심화, 스프링 데이터 JPA, 트랙잭션
    
5. **[주니어 백엔드 개발자가 반드시 알아야 할 실무지식]**
   - 기간: 2025.07 ~ 2025.12
   - 주요 내용: 성능 병목, DB 쿼리, 보안, 네트워크 등 실무에서 마주치는 핵심 이슈

6. **[스프링6와 스프링 부트3로 배우는 모던 API 개발]**
   - 기간: 2025.12 ~ 2026.00
   - 주요 내용: REST, gRPC, GraphQL 및 비동기 API, 워크플로 및 오케스트레이션 엔진을 사용한 마이크로서비스 기반 솔루션의 개발

7. **[요즘 당근 AI 개발]**
   - 기간: 2025.12 ~ 2026.00
   - 주요 내용: 당근 실무자들의 AI 사용법
